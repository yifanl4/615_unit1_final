Exploratory data analysis into the chemicals being applied to blueberries

Acquire and read the data

These data were collected from the National Agricultural Statistics Service, which is the statistical branch of the U.S. Department of Agriculture. According to its website, NASS conducts hundreds of surveys and issues nearly 500 national reports each year on issues like agricultural production, economics, demographics, and the environment. The data were collected from the USDA database selector: https://quickstats.nass.usda.gov which were mainly about the information about berries.

The data were stored online and then downloaded as a CSV file.

The berries data only had 8 out of 21 columns containing meaningful data. Thus further data cleaning methods are required. Below is a preview of the first parts of the original data.

Data Cleaning and Organization

Method

  • First filter out a dataset names bberry which contains all the blueberries data from the original dataset. Then separate the Data Item column into several different columns so that further cleaning process can be conducted.

  • Similarly, separate the Domain and the Domain Category columns for further cleaning process.

  • The next step is to find and remove the redundant columns.

  • Then the what and the means columns need to be cleaned and re-organized. The objective is to put these two columns all in the same column. In order to do so, separation is needed first and then test to make sure there are no overlaps.

  • Finally rename the columns of the dataset and select those that are required in this analysis.

We first create a box plot of chemical values being applied to blueberries according to state.

According to the box plot, we can easily tell that there are a large number of outliers for each state. Thus we try to create another one without the outliers for visualizing the distribution of each state.

We can repeat the same procedure to visualize the distributions of chemicals being applied to blueberries according to different types of chemical.

According to the box plot we can discover that there appears to have two different groups of chemicals with extremely different ranges of values. Thus we can divide our unfood_1 dataset into two sets according to there two different groups of chemicals and then plot.

Then we can create a histograms to see the least and most variance chemicals. According to the histogram, we can see that nearly 300 out of 552 brands of blueberries have values of chemicals below 0.5. However, there is still a wide distribution of the values of chemicals contained by selected blueberries due to the varieties of chemicals which require further analysis.

Conclusion

After the basic exploratory data analysis, we could come out with these conclusions:

  • The original data is messy and contains numerous NA values which make it inappropriate for data analysis. Thus data cleaning is a necessary step in this situation.
  • Even after data cleaning, there are still many outliers exist which require further analysis before doing any advanced cleaning steps.
  • From the plot of chemical values according to state, we can tell that it appears New Jersey has the most various chemical values while Washington has the least various chemical values.
  • From the plot of chemical values according to types of chemicals, we can tell that it appears there are two different groups measured chemicals which need to be treated separately in further analysis.
  • From the histogram of chemical values, we can tell that it appears most (nearly 300 out of 552) brands blueberries contain chemical values below 0.5.
